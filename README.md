# AR的当てゲーム
## 概要
- ランダムな位置に的になる立方体が生成される
  - 立方体はプレイヤーに向かってくる
- プレイヤーは画面タップでボールを撃てる
- 的が自分に当たる前にボールを当てて壊す
  - 的が自分に当たってしまうとゲームオーバー
